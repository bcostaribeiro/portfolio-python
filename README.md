## 🛠 Tech Stack / Tecnologias

* **Language:** Python 3.x
* **Core Libraries:** `Requests` (HTTP), `BeautifulSoup4` (HTML Parsing), `NewsAPI`.
* **Key Skills:** Web Scraping, API Integration, Data Logic, and Automation Architecture.

---

## 📂 Project Structure / Estrutura do Projeto

### 📰 `/News` (News Intelligence)
*Focus: Automated data extraction from major news portals.*

| Script | Function (EN) | Função (PT) |
| :--- | :--- | :--- |
| `br_news_aggregator.py` | Central engine for scraping G1, R7, and CNN. | Motor central de scraping (G1, R7, CNN). |
| `global_news_api_client.py` | Real-time global news via API integration. | Notícias globais via integração de API. |
| `cnn_brazil_scraper.py` | Targeted scraper with local `.txt` export. | Scraper especializado com exportação `.txt`. |
| `g1_brazil_scraper.py` | High-precision scraping using CSS classes. | Scraper de alta precisão via classes CSS. |

### ⚙️ `/Others` (Automation & Logic)
*Focus: General process automation and market data tools.*

* `ColetorDadosDeMercado.py`: Market data collection tool.
* `MonitorPrecoIphone.py`: Automated price tracking for smartphones.
* `BuscadorNoticiasDoYouTube.py`: Automated YouTube content searcher.
* `FiltroDeRecrutamento.py`: Logic-based tool for candidate filtering.
* `moderadorcomentario.py`: Automatic comment moderation script.

## 💻 How to Run / Como Executar

1. **Install dependencies / Instalar dependências:**
   
   pip install requests beautifulsoup4

2. **Run the aggregator / Executar o agregador:**

python News/br_news_aggregator.py
